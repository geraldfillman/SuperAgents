# Title Scoring Skill

## Purpose

Apply a repeatable scorecard to titles rather than chasing studio narratives.

## Scoring dimensions

- Production visibility
- Funding resilience
- Launch readiness
- Storefront momentum
- Community quality
- Critic upside
- Portfolio dependence
- Post-launch monetization
- Execution risk
- Disclosure quality

## Scripts

### `scripts/score_title.py`

Score a single title from input JSON or run a small batch scoring pass.
