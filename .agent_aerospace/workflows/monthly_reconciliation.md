# Monthly Reconciliation Workflow

1. Reconcile contract records against the latest saved source set.
2. Update budget-line assumptions for the next fiscal window.
3. Refresh runway estimates using the latest quarterly filings.
4. Mark stale milestones and archive invalidated assumptions.
5. Publish a monthly operating report with:
   - active systems
   - upcoming milestones
   - top funding dependencies
   - top financing risks
