# Daily Update Workflow

1. Pull new contract award records or announcement manifests.
2. Refresh SAM opportunities, SBIR awards, and FAA commercial-space pages for early-stage names.
3. Check new 8-K, 10-Q, and 10-K filings for procurement language.
4. Add or update test-event evidence for active systems.
5. Refresh the 90-day program calendar.
6. Flag source confidence for every new record.
7. Emit a daily summary of:
   - new awards
   - new pre-award signals
   - new SBIR traction
   - new FAA licensing or stakeholder signals
   - new milestone dates
   - slippage or cancellation risk
   - low-runway companies with pending milestones
