# MiroFish Cross-Sector Signal Cascade Demo

Date: 2026-03-15

## Scenario

Test the `Cross-Sector Signal Cascades` idea with a three-agent MiroFish graph:

- aerospace trigger agent
- rare earth constraint agent
- renewable energy cost-risk agent

The simulation seeds one upstream supply-chain warning and observes whether the discussion
propagates across adjacent sector lenses.

## Inputs

- spec: `docs/simulation_specs/cross_sector_signal_cascade.yaml`
- bundle: `data/processed/mirofish_simulations/sim_cross_sector_signal_cascade/`
- runtime: `_reviews/MiroFish/`
- python: `.venv-mirofish/Scripts/python.exe`

## Run Command

```powershell
.venv-mirofish\Scripts\python.exe .agent_simulation\skills\mirofish_runtime\scripts\run_bundle.py `
  --bundle-dir data\processed\mirofish_simulations\sim_cross_sector_signal_cascade `
  --platform parallel `
  --max-rounds 8 `
  --no-wait `
  --json
```

## Outcome

- bundle creation succeeded
- runtime probe was healthy in `.venv-mirofish`
- parallel simulation completed with exit code `0`
- twitter action log recorded `12` total actions
- reddit action log recorded `12` total actions

## Visible Cascade Behavior

The simulation produced the intended shape:

- round 0 seeded all three starting posts
- later rounds showed cross-sector follow-on behavior instead of only the original seed posts
- the renewable-energy agent quoted the aerospace trigger and reframed it as upstream project risk
- the rare-earth agent amplified the processing-capacity interpretation
- both platforms produced non-empty logs and databases

- the dedicated `.venv-mirofish` environment was required; the default Python 3.14 environment is

  prevent simulation completion

## Best Next Iteration

Use this same graph structure, but replace the hand-written seed posts with outputs generated by
real Super Agents scripts:

1. aerospace writes the trigger
2. rare earth consumes that trigger and writes a structured response
3. renewable energy reacts only if the rare earth confidence exceeds a threshold

That would move the demo from persona-only simulation into true agent-to-agent choreography.
